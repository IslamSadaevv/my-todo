# Todo • Pro — React + TypeScript Todo App

Eenvoudige, maar professionele Todo-app gebouwd met React + TypeScript.  
Laat zien: UI/UX ontwerp, state-management, persistente opslag (localStorage), en systematisch testen.

## Features
- Taken toevoegen, bewerken (dubbelklik of potlood), verwijderen
- Prioriteit (low/medium/high) en due date per taak
- Filters (Alle / Actief / Voltooid), zoeken en sorteren
- Clear completed met bevestiging
- Persistente opslag via localStorage
- Volledig responsive en toegankelijk (aria-labels, focus states)

## Run locally
1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173 (of URL die Vite toont)

## Build
`npm run build` — voor productie

## Tech
- React (Vite template)
- TypeScript
- CSS (pure)
